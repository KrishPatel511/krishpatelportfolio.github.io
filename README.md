## ⚡ Quick Start

```bash
# Clone the repo
git clone https://github.com/KrishPatel511/krishpatelportfolio.github.io.git

# Install dependencies
cd krishpatelportfolio.github.io
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## ☁️ Deployment

Deployed on **Vercel** with auto-deploy on every push to `main`.

1. Push code to GitHub
2. Import repo at [vercel.com](https://vercel.com)
3. Click **Deploy** — done!
